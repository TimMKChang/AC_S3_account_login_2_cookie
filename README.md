## Installing
1. clone the project
>git clone https://github.com/TimMKChang/AC_S3_account_login_1.git
2. install packages
>npm install
3. run the server
>npm run dev
4. check in cmd
>Now is listening on [localhost:3000](http://localhost:3000)
